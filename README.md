# build-search

Learn search engine algorithms and system design through guided Java exercises.

## Run In IntelliJ IDEA

Use the shared run configurations:

- `Searchlings - Default`
- `Searchlings - List`
- `Searchlings - Stage 01`
- `Searchlings - Stage 02`
- `Searchlings - Stage 03`
- `Searchlings - Stage 04`

The stable click-to-run entrypoint is:

- `me.catcodeme.search.app.Searchlings`

## Project Map

- `src/main/java/me/catcodeme/search/app`: application entrypoint and command dispatch.
- `src/main/java/me/catcodeme/search/core`: search models, tokenization, and algorithms.
- `src/main/java/me/catcodeme/search/corpus`: sample documents and future data loaders.
- `src/main/java/me/catcodeme/search/exercises`: guided lessons and exercise catalog.
- `.run`: shared IntelliJ IDEA run configurations that should be committed.
